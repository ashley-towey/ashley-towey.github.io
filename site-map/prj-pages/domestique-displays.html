Project Description

Domestique displays is a project which aims to increase engagement in the sport of road cycling through the introduction of a matrix led screen which acts as a bridge between the racers and the spectators. Communicating the live race data to attendees, and giving the spectators the ability to add a supportive animation back to the riders. Additionally a web application helps to build excitement before the event, supporting functionality during the event, and post-race analysis to engage with supporters and racers before, during, and after events.

This project is a work in progress, and will be on display at the 2024 CSM Graduate Showcase between the 11th and 16th of June.

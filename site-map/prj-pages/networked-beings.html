Project Description

Three networked objects; Bool, Boid, and Buzz explore how our interaction with objects alters our desire to interact with them. Through micro-interactions of turning each others light on they create a connection. Each object is created in a different form which relates directly to it’s output device. Bool is sharp outputting 1 or 0, and turning the affected device on or off. Boid is square, and has four outputs, each relating to a different colour state on the affected device. Buzz is circular and infinite, outputting an ever-changing random array of colours.
